В процессе ручного тестирования приложения были выявлены следующие недочеты:
1) При переходе по ссылке для скачивания CentOS происходит переход на скачивание Fedora.
2) При попытке зарегистрировать пользователя с несколькими невалидными полями всплывающее уведомление содержит сообщение об ошибке в некорректном виде.

В процессе автоматизированного тестирования приложения были выявлены следующие ошибки:
1) UI: Ошибка при попытке зарегистрировать пользователя с паролем, превышающим максимальную длину поля в базе данных.
Отображается Internal Server Error вместо Incorrect password length

2) API: Ошибки при попытке авторизации пользователя с некорректной длиной имени, пустым полем для имени, пустым полем для пароля, пустыми полями для имени и для пароля.
Возвращается код ответа 200 вместо ожидаемого 401.

3) API: Ошибка при загрузке главной страницы.
Для загрузки главной страницы запрашивается элемент, который возвращается с кодом 404 вместо ожидаемого 200.

4) API: Ошибка при попытке зарегистрировать пользователя с паролем, превышающим максимальную длину поля в базе данных.
Возвращается код ответа 500 вместо ожидаемого 400.

5) API: Ошибка при попытке зарегистрировать пользователя с некорректным адресом электронной почты.
Возвращается код ответа 200 вместо ожидаемого 400.

6) API: Ошибка при попытке зарегистрировать пользователя с уже существующим именем.
Возвращается код ответа 409 вместо ожидаемого 400.

7) API: Ошибка при попытке зарегистрировать пользователя с уже существующим адресом электронной почты.
Возвращается код ответа 500 вместо ожидаемого 400.

8) API: Ошибка при попытке добавления пользователя с уже существующим адресом электронной почты.
Возвращается код ответа 210 вместо ожидаемого 400.

9) API: Ошибка при попытке добавления пользователя с уже существующим именем.
Возвращается код ответа 304 вместо ожидаемого 400.

10) API: Ошибка при попытке добавления пользователя.
Возвращается код ответа 210 вместо ожидаемого 400.

11) API: Ошибка при попытке отправить неверный запрос для добавления нового пользователя.
Возвращается код ответа 405 вместо ожидаемого 404.
